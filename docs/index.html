GWSFluid simplifies creating fluid UI responsive Android Apps.

Speeds up development

Most of the heavy lifting has been done already for you to support your MVVM or MVP android app architecture patterns. Full support of the Google de-coupled Presenter style supported.

We keep track of the differences in the UI and graphics pipe-line per Android versions so you can focus on your cool Android app.

Easy to work with

We have provided detailed documentation as well as specific code examples to help new users get started. We are also always open to feedback and can answer any questions a user may have about Materialize.
